# AI Knowledge Vault

A Flask-based RAG (Retrieval-Augmented Generation) system using:

- OpenAI GPT-4o-mini
- OpenAI Embeddings (text-embedding-3-small)
- FAISS vector database
- dotenv for secure API management

## Setup

1. Clone repo
2. Create virtual environment
3. Install requirements
4. Add .env file with OPENAI_API_KEY
5. Run python app.py
